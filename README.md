Credit Card Fraud Detection System
📝 Project Overview
This project is an Artificial Intelligence solution designed to identify fraudulent credit card transactions. Using historical transaction data, the model classifies transactions as either "Normal" or "Fraudulent," helping to prevent financial theft in real-time.

🚀 Features
Data Preprocessing: Handling large datasets to prepare them for machine learning models.

Model Training: Implemented RandomForestClassifier to achieve a high accuracy rate.

Interactive Dashboard: Built using Streamlit to visualize data and show model predictions in a user-friendly interface.

High Accuracy: The model currently performs with 99.51% accuracy on the test dataset.

🛠️ Technologies Used
Language: Python

Libraries: Pandas, NumPy, Scikit-Learn, Streamlit

Environment: VS Code

Deployment: Localhost (Streamlit Server)
